Config Loader¶

Module that provides convenience for loading data from a YAML configuration file.

config_loader¶

The config_loader ensures that additional languages can be supported by simply modifying the config.yaml file.

hola_mundo.util.config_loader.get_config(config_key: str) → [~AnyStr]¶

Retrieves the configuration list for a given config_key.

Parameters:config_key – The key to that target configuration list.
Returns:A configuration list for the given configuration key.

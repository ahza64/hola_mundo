ElGrito¶

ElGrito is a great way to get translations for ‘Hola’.

class hola_mundo.el_grito.ElGrito¶

Class that produces a random language version of ‘hola’.

grito(index: int=None) → str¶

Method to retrieve a language version of ‘hola’.

Parameters:index – An optional index key to retrieve a given ‘hola’ translation. If no index key is provided, a random translation of hello is generated.
Returns:A language version of ‘hola’.
Except:IndexError - Thrown if there is no language translation for the given index.

ElMundo¶

ElMundo is a great way to get translations for Mundo.

class hola_mundo.el_mundo.ElMundo¶

Class that produces a random language version of ‘mundo’.

mundo(index: int=None) → str¶

Method to retrieve a language version of ‘mundo’.

Parameters:index – An optional index key to retrieve a given ‘mundo’ translation. If no index is set, then a random translation of world is returned.
Returns:A language version of ‘mundo’.
Except:IndexError - Thrown if there is no language translation for the given index.
